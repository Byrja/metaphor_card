# UX Regression Notes v3

## Потенциальные риски

### По тону
- Уточнение формулировок может в некоторых случаях звучать менее выразительно
- Возможна потеря части эмоциональной насыщенности отдельных формулировок
- Некоторые изменения могут повлиять на восприятие последовательности взаимодействия

### По безопасности
- Все изменения сохраняют безопасный тон взаимодействия
- Не внесены изменения в кризисные формулировки и поддержку
- Механизмы безопасности остались без изменений

## Рекомендации по тестированию
- Проверить восприятие новых формулировок в контексте полного сценария
- Убедиться в сохранении интуитивности переходов между этапами
- Проверить адекватность "маленьких шагов" в различных контекстах

## Обратная связь
- Рекомендуется собрать обратную связь от пользователей по новым формулировкам
- Особое внимание уделить восприятию изменений в кризисных сценариях
- Проверить эффективность новых формулировок в поддержании мотивации
