# UX Changelog v3

## Улучшения UX и copy

### /situation (Анализ ситуации)
- Уточнены формулировки вопросов для более бережного тона
- Адаптированы поддержки для соответствия стилю остального приложения

### /checkin (Быстрый чек-ин)
- Улучшены формулировки вопросов о теле и ощущениях
- Адаптированы поддержки для более мягкой формулировки

### /day (Карта дня)
- Уточнены формулировки итогов и "маленьких шагов"
- Сделаны более персонализированными и поддерживающими

### Microcopy ошибок
- Улучшены сообщения об ошибках для более бережного взаимодействия
- Сделаны менее техническими и более человечными

## Общие улучшения
- Все изменения направлены на создание более бережного и поддерживающего тона
- Устранены возможные директивные формулировки
- Улучшена связность между различными частями приложения

## Цель изменений
- Создать более мягкую и поддерживающую атмосферу для пользователей
- Усилить ощущение личного сопровождения в процессе работы с картами
- Сделать взаимодействие более интуитивным и естественным
