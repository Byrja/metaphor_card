# Кнопочная навигация Metaphor Card (MVP)

## Главное меню

### Основные функции
- 🃏 **Карта дня** - Получить метафорическую карту дня
- 💭 **Быстрый чек-ин** - Отметить свое текущее состояние
- 🎯 **Разбор ситуации** - Работа с конкретной жизненной ситуацией
- 📔 **Личный дневник** - Просмотр сохраненных инсайтов

## Сценарий "Карта дня"

### Начало
- 🃏 **Получить карту** - Сгенерировать карту дня

### После получения карты
- 🔍 **Углубиться в смысл** - Получить наводящие вопросы
- 💾 **Сохранить инсайт** - Записать свои мысли по карте
- 🏠 **В меню** - Вернуться в главное меню

### Углубление в смысл
- После выбора "Углубиться в смысл" пользователь получает наводящие вопросы из промптов

### Сохранение инсайта
- После выбора "Сохранить инсайт" пользователь вводит текст инсайта
- Затем подтверждает сохранение или отменяет

## Сценарий "Быстрый чек-ин"

### Начало
- 💭 **Начать чек-ин** - Ответить на начальный вопрос

### В процессе чек-ина
- После ввода текстового ответа пользователь выбирает эмоцию:
  - 😊 **Радость**
  - 😢 **Грусть**
  - 😠 **Гнев**
  - 😨 **Страх**
  - 🤷 **Другое**

### Завершение
- 🏠 **В меню** - Вернуться в главное меню

## Сценарий "Разбор ситуации"

### Начало
- 🎯 **Описать ситуацию** - Ввести текстовое описание ситуации

### Работа с ситуацией
- После ввода описания показывается первая карта
- 🃏 **Следующая карта** - Перейти к следующей карте в анализе
- 🏁 **Завершить анализ** - Закончить сессию разбора

### Завершение
- 🏠 **В меню** - Вернуться в главное меню

## Личный дневник

### Просмотр
- При выборе "Личный дневник" отображаются последние 5 сохраненных инсайтов
