# BUTTON_MAP_MVP

## Главное меню
| Кнопка | Команда | Назначение |
|---|---|---|
| Карта дня | `/day` | Быстрый вход без длинной сессии |
| Чек-ин | `/checkin` | Короткая self-check с опорой |
| Разобрать ситуацию | `/situation` | Основной сценарий на 3 карты |
| История | `/history` | Посмотреть сохранённые инсайты |
| Повторяющиеся темы | `/patterns` | Показать темы и подготовить `/nudge` |
| Мягкая подсказка | `/nudge` | Дать аккуратный follow-up |

## Локальные действия в сценариях
| Контекст | Кнопка | Команда/действие |
|---|---|---|
| Любой сценарий | Пауза | future callback / pause |
| Любой сценарий | Завершить | future callback / stop |
| После инсайта | Сохранить инсайт | `/insight <text>` |
| После паттернов | Взять подсказку | `/nudge` |

## Примечание по runtime
В текущем runtime остаются slash-команды. Таблица выше — прямое ТЗ для следующего UI-слоя без ручной интерпретации.
