# SESSION_COPYBOOK_MVP

## /start
> Привет. Это пространство для короткой и бережной саморефлексии через метафорические карты.  
> Без гадания, без диагнозов, без давления.

## /day
- Заголовок: `Карта дня: {title}`
- Вопрос: prompt из L1
- Выход: не перегружать, без длинной интерпретации

## /checkin
- Заголовок: `Сделаем короткий чек-ин без спешки.`
- Подводка: `Карта опоры: {title}`
- Дальше: 3 коротких prompt-а L1 → L2 → L4

## /situation
- Заголовок: `Разбор ситуации в 3 шага.`
- Карты:
  - `Карта 1 — где ты сейчас.`
  - `Карта 2 — что мешает.`
  - `Карта 3 — что поможет.`
- Финал:
  - `Углубляющий вопрос: {prompt}`
  - `Когда появится формулировка, сохрани её через /insight <текст>.`

## /history
- Пусто: `Пока здесь пусто. Первый инсайт можно сохранить через /insight <текст>.`
- Список: `Твои последние инсайты:`

## /patterns
- Пусто: `Пока данных мало. Сохрани ещё 2–3 наблюдения, и я соберу повторяющиеся темы.`
- Список: `Повторяющиеся темы в последних инсайтах:`
- CTA: `Если хочешь, можно взять мягкую подсказку через /nudge.`
