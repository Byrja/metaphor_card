# UX_TEXT_GUIDE

## Принципы MVP
- Тон: тёплый, ясный, без эзотерики и без терапевтических обещаний.
- Один экран — одна задача.
- Команда звучит как приглашение, а не приказ.
- В каждой ветке есть понятный выход: пауза, завершение, маленький шаг.

## Базовые формулы
- Вместо «Команды» → «Что можно сделать сейчас».
- Вместо «сохрани инсайт» → «если захочешь, можно сохранить».
- Вместо «не понимаю» → «я лучше всего понимаю команды, можно начать с /start».

## Лексика
- Используем: «заметить», «отклик», «маленький шаг», «без спешки», «опора».
- Избегаем: «предсказание», «судьба», «истина о тебе», «ты должен».

3. Вторичные действия — ниже.
