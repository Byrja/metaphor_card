# Руководство по стилю текстов Metaphor Card

## Общий стиль и тон

### Основные принципы
- **Бережность**: Все тексты должны создавать ощущение заботы и внимания
- **Интимность**: Тексты должны быть как личный разговор с близким человеком
- **Простота**: Избегать сложной терминологии, объяснять профессиональные термины
- **Честность**: Не обещать того, чего нет, не делать завышенных заявлений

### Тон общения
- **Соучастие, а не наставничество**: "Давай разберемся вместе", а не "Я знаю, что для вас лучше"
- **Интерес, а не суждение**: "Расскажи подробнее", а не "Это неправильно"
- **Поддержка, а не давление**: "Ты уже сделал важный шаг", а не "Ты должен это сделать"

## Антипаттерны

### Что НЕЛЬЗЯ делать
- Не использовать обесценивающие формулировки ("Это же очевидно", "Ну это же просто")
- Не давить на пользователя ("Ты обязан", "У тебя нет выбора")
- Не использовать профессиональный жаргон без объяснения
- Не делать предположения о пользователе ("Ты наверняка чувствуешь...")
- Не использовать шаблонные фразы ("Все будет хорошо", "Это нормально")

### Что НЕ рекомендуется
- Избегать слишком формального тона
- Не перегружать текст терапевтическими терминами
- Не использовать клише и штампы
- Не делать акцент на проблемах, а не на решениях

## Стилистические рекомендации

### Язык
- Использовать активный залог
- Предпочитать короткие предложения
- Использовать обращение на "ты"
- Избегать пассивного залога и сложных конструкций

### Эмодзи и оформление
- Умеренное использование эмодзи для создания теплой атмосферы
- Жирный шрифт для акцентов
- Списки для структурирования информации
- Подзаголовки для навигации

## Живые формулировки

### Вместо "всё хорошо" используй:
- "Ты уже сделал важный шаг"
- "Ты справишься"
- "Ты не один"
- "Это нормально - чувствовать так"

### Вместо "баланс" используй:
- "Пауза может помочь"
- "Давай сделаем глубокий вдох"
- "Ты имеешь право на отдых"
- "Ты можешь быть бережнее к себе"

### Вместо "поддержка" используй:
- "Я рядом"
- "Ты имеешь право на заботу"
- "Ты не один с этим"
- "Ты заслуживаешь внимания"

## Конкретные примеры

### Приветствия:
- "Привет! Готов к небольшому путешествию внутрь себя?"
- "Рад тебя видеть! Что сегодня с тобой происходит?"
- "Спасибо, что нашел время для себя"

### Переходы:
- "Давай продолжим"
- "Что дальше?"
- "Хочешь пойти глубже?"
- "Есть еще один шаг"

### Завершения:
- "Спасибо за доверие"
- "Ты проделал важную работу"
- "Это уже забота о себе"
- "Ты заслуживаешь этих открытий"
